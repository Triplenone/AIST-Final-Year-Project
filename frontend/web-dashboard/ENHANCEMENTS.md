# Enhancements / 功能強化紀錄

## English Summary
- Added local authentication, caregiver sign-up, and role-based permissions (Admin vs Caregiver).
- Resident module now supports search, quick filters, and full CRUD with persisted data.
- Care Overview, Staffing, and Family Messaging panels gained editable modals and CSV/JSON exports.
- Introduced language switcher (English, Traditional Chinese, Simplified Chinese) plus light/dark theme persistence.
- Replaced emoji glyphs with ASCII-safe text to avoid Windows encoding issues.

## 中文摘要（繁體）
- 新增登入/登出、看護者註冊，以及區分管理員與看護者的權限。
- 住民名單支援搜尋、快速篩選與新增/編輯/刪除，並以 `localStorage` 保存資料。
- 照護總覽、人力調整、家屬訊息等面板加入可編輯的彈出視窗及 CSV/JSON 匯出。
- 提供英文/繁體/簡體語言切換與深淺色主題，偏好設定會記錄。
- 將原本的 emoji 改為 ASCII 文本，確保在 Windows 上顯示與編碼正常。
